# Open-source foundations

## Alex OS window system

- Architecture reference: [Dragon-Born/MacScape](https://github.com/Dragon-Born/MacScape), MIT licensed.
- Interaction dependency: [bokuweb/react-rnd](https://github.com/bokuweb/react-rnd), MIT licensed.
- Interface icons: [Phosphor Icons](https://github.com/phosphor-icons/react), MIT licensed.

The implementation keeps the site’s own components, file model, media, and visual system. MacScape informed the small window-manager structure; `react-rnd` supplies the maintained drag-and-resize behavior. No Apple-owned icon or operating-system asset bundle is redistributed.

## Content boundaries

- Work contains finished portfolio case studies.
- Play contains smaller but still presentable design projects.
- Music, video sketches, raw experiments, and deeper archive material belong in Alex OS.
